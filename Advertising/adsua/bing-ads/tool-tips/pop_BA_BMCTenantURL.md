# Tenant URL

**What it is:**  The URL a Microsoft Advertising tenant uses to access their [Content API](https://go.microsoft.com/fwlink?LinkId=404097) services. This URL is a combination of the service endpoint and the tenant's Microsoft Merchant Center store ID.

**Example:**  http://content.api.bingads.microsoft.com/shopping/v9.1/bmc/{bmcMerchantID}
